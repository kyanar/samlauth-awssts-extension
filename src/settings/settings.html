General Options

Options on this page configure general extension settings that affect all accounts.

Account Options

This page allows you to map account IDs to "friendly names" which will be saved in your credentials file as profiles. Note that after changing this, you will need to re-authenticate to any accounts you have added a new friendly name for in order for this change to take effect.
